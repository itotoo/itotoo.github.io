# http

1. HTTP协议即超文本传送协议
2. HTTP连接是一种“短连接”；
3. 在HTTP 1.0中，客户端的每次请求都要求建立一次单独的连接，在处理完本次请求后，就自动释放连接。
4. 受影响因素，带宽和延迟，dns查询，浏览器请求数阻塞4-6，线头阻塞
